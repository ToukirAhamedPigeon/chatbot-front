## üíª Local Setup

1.  **Navigate to the directory**:
    ```bash
    cd D:\chatbot\chatbot-front
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Configure Environment**:
    Create or edit the `.env` file to point to your backend API.
    ```env
    VITE_API_URL=http://localhost:8000
    ```

4.  **Run the Development Server**:
    ```bash
    npm run dev
    ```
    The app will be available at `http://localhost:5173`.

## üì¶ Build for Production

## üîå API Integration

The frontend communicates with the backend via the `POST /chat` endpoint.

**Request Payload:**
```json
{
  "question": "User input",
  "filters": {
    "topic": "Selected Topic",
    "difficulty": "Selected Difficulty"
  }
}
```

**Response Handling:**
- Displays the `answer` in the chat bubble.
- Shows `sources` (if available) below the answer for context.
- Handles loading states and error messages gracefully.

## üé® UI/UX Details

- **Topic Badges**: Visual indicators for the active topic.
- **Auto-Scroll**: Chat window automatically scrolls to the latest message.
- **Loading Indicators**: "Typing..." animation while waiting for the backend response.
- **Error Handling**: Friendly error messages if the backend is unreachable.

## ‚ùì Troubleshooting

- **Backend Connection Error**: Ensure the backend server is running on port 8000 and CORS is enabled (or the proxy is configured).
- **`npm install` fails**: Delete `node_modules` and `package-lock.json`, then try again.
